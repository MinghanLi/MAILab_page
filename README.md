# MAI Lab

Static multi-page homepage scaffold for **MAI Lab: AI for Visual Intelligence and Engineering**.

## Local preview

```bash
python3 main.py
```

Then open `http://127.0.0.1:8000`.

## GitHub Pages

This repo is structured to work as a simple GitHub Pages site because the
homepage and subpages are served from static `.html` files in the repo root.

Recommended repository name for an organization site:

- `mai-lab.github.io`

Recommended sections to expand next:

- `People`
- `Teaching`
- `Projects`
- `Publications`
- `News`
- `Resources`
- `Openings`
